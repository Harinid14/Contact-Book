Simple Contact Book

Objective:

Develop a contact book application that allows users to store, view, search, and delete their contacts.

Steps:

Create an interface for adding a new contact with fields for name, phone number, and email.

1. Store the contacts in a data structure (e.g., an array or object).
2. Implement a feature to display a list of all stored contacts.
3. Allow the user to search for contacts by name.
4. Display matching results.
5. Add a feature to delete a contact from the list.
6. Test the application to ensure all features (adding, viewing, searching, and deleting) work as expected.
